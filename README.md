# ECG Signal Analyzer App

Interactive Streamlit app for visualizing ECG signals, applying bandpass filtering, detecting R-peaks, and estimating heart rate.

---

## Biomedical Context

Electrocardiography (ECG) is a fundamental tool for assessing cardiac rhythm and conduction. This app is designed as an educational tool for biomedical engineering students to explore how filtering and peak detection algorithms affect ECG interpretation and heart rate estimation.
